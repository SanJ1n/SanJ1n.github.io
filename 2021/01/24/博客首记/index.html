0x01 安装

安装的流程是首先将博客在本地部署还有调试好后在将其推到github上，这里就节省了购买服务器的钱（能嫖就嫖），当然如果你介意域名有github或者想用自己的机器，也可，这样直接节省了向github上推的步骤了，废话不多说，开始部署，以下的很多的命令还是在本地进行配置的。

第一步 安装node.js

首先是在本机进行安装，由于本人用的虚拟机是kali，所以这里直接使用 apt-get 安装比较方便。

添加 -v参数 查看版本如果都可以看到说明安装成功

第二步 跟换源

这里由于npm使用的是国内源，所以这里需要换到国内淘宝源

npm install -g cnpm –registry=https://registry.npm.taobao.org

注意的是其报的warm这里是不需要太在意的，如图已经查看版本说明安装成功

第三部 安装hexo博客

利用cnpm进行创建博客

cnpm install -g hexo-cli

查看版本，说明安装成功

第四步

创建一个专门用于放置blog所用的文件，并启动博客

mkdir /home/blog cd /home/blog sudo hexo init /home/blog 初始化博客 hexo s

出现如图表示初始化成功

此时表示博客已经运行

访问一下，可以访问成功

hexo部署成功

将本地hexo部署到github上

登录后新建一个github仓库

填写仓库名称填写说明，点击创建仓库，便常见成功

如果要将本地文件部署到github上去，还需要在本地下载一个插件

cnpm install –save hexo-deployer-git
